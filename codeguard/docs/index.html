🛡️ CodeGuard

Ethical AI Code Review Agent for private GitHub repositories

🤖 LangGraph ⚡ Groq LLaMA3 🔒 Human-in-the-loop 🐍 Python 3.11+

⚡ Quick Install

curl -fsSL https://omthakur1394.github.io/codeguard/install.cmd -o install.cmd && install.cmd && del install.cmd
curl -fsSL https://omthakur1394.github.io/codeguard/install.sh | bash

✨ Features

🔍 OSS Detection

Automatically detects open-source repos and refuses to clone or review them.

🤖 AI Review

Reviews every file for bugs, security issues, style problems and suggestions.

🔧 Auto Fix

Generates production-ready fixed code for every file it reviews.

👀 Human Approval

Shows diff for every change. You approve or reject each file individually.

🔐 Ownership Verify

Verifies you are the repo owner before allowing any commits or pushes.

🚀 Zero Storage

GitHub token never stored anywhere. Lives in memory only for the session.

Built with ❤️ by Om Thakur

CodeGuard never reviews what it shouldn't. And never pushes what you haven't approved.
